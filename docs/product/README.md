# Product Strategy

TheHood is a deterministic local runtime with visible premium model escalation. The runtime owns loop control, permissions, evidence, approvals, artifacts, and role separation. ChatGPT Pro is a strategic judgment layer for work where better reasoning changes the outcome.

The product promise is not hidden Pro usage. The promise is governed execution with deliberate escalation when the task is ambiguous, high-value, reputational, or blocked by disagreement.
