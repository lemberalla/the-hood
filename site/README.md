# TheHood Static Site

This directory is the static public-preview website for TheHood.

It is intentionally dependency-free:

- no backend
- no analytics
- no client-side build step
- no runtime secrets

Open `site/index.html` directly in a browser, or publish the directory with GitHub Pages.

The repository docs remain the source of truth. Site pages should summarize current public-preview behavior and link back to canonical docs instead of inventing separate product claims.
