Known Limitations

The preview is useful for early adopters, but several surfaces remain experimental or future work.

AreaCurrent boundary
Hosted executionNot included. Runs are local-session workflows.
SchedulesNot included. No timers, daemons, or cloud queues.
API providersOpenAI and Anthropic API metadata exists, but production adapters are not wired.
ChatGPT MCP connectorExperimental and dependent on external ChatGPT workspace availability.
Native appFuture thin control surface. The CLI and MCP server are the current primary surfaces.
Public packagePrepared for preview publishing, but release still requires final audit and tag.
Source docs: `docs/KNOWN_LIMITATIONS.md` and `docs/PUBLIC_REPO_READINESS.md`.
